# RODRIGO.DEV | Especialista em Dados & Automação

Este é o portfólio profissional de **Rodrigo Vieira**, um desenvolvedor e analista focado em transformar dados complexos em soluções de automação de alto impacto.

## 🚀 Tecnologias
- **Frontend:** React, Tailwind CSS, Framer Motion
- **Backend/Automação:** Python, SQL, Express
- **Design:** Modern UI/UX, Dark Mode Industrial

## 📦 Como Rodar Localmente
1. Clone o repositório:
   ```bash
   git clone https://github.com/Devolprodrigov/rodrigo-dev-portfolio.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 🌐 Deploy (GitHub Pages)
O portfólio já está configurado com **GitHub Actions**. Para que ele pare de ficar em branco:
1. Vá nas **Settings** do seu repositório no GitHub.
2. Clique em **Pages** (na barra lateral esquerda).
3. Em **Build and deployment** > **Source**, mude para `GitHub Actions`.
4. O site será compilado e publicado automaticamente a cada novo push na branch `main`.
